# CLAUDE.md

This file is an **adapter**. The canonical agent guidance lives in [`AGENTS.md`](AGENTS.md) —
read it first (constitutional hard stop, access rules, invariants, runbooks, operating notes).

Operational procedures are **spec-as-code runbooks** in `vault-template/96-Runbooks/`
(e.g. `close-daily`, `seal-provenance`). To perform one, open and follow the runbook; don't
improvise. See ADR-0011 (spec-as-code runbooks) and ADR-0012 (daily close lifecycle).
